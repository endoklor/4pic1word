🎮 4 Фото 1 Слово

Простая и увлекательная игра-головоломка, в которой нужно угадать слово по четырём изображениям.

📌 Описание

Игроку показываются 4 картинки, объединённые одной идеей. Задача — понять, что их связывает, и составить правильное слово из предложенных букв.

Игра состоит из уровней, сложность которых постепенно увеличивается.

🚀 Возможности
20 уровней с возрастающей сложностью
Система подсказок
Подсказки открывают буквы в случайном порядке
Простой и понятный интерфейс
Подходит для тренировки логики и ассоциативного мышления
💡 Как играть
Посмотрите на 4 изображения
Определите, что их объединяет
Соберите слово из предложенных букв
Используйте подсказки, если возникли трудности
🔧 Подсказки
Открывают одну случайную букву
Работают только с ещё не открытыми буквами
Помогают пройти сложные уровни
📁 Структура проекта

(пример — при необходимости измени под свой проект)

main.py — основной файл с кодом игры
install.py — установка нужных библиотек
run.py — запуск игры
assets/ — изображения уровней

🛠️ Установка и запуск
Скачайте или клонируйте проект
Откройте файл run.py в командной строке написав "python run.py"
